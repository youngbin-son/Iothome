# Iothome
## 설명 
아두이노와 라즈베리를 이용한 Iothome
모바일 폰으로 LED, FAN 조작가능,
온습도,조도 센서값을 SQL에 등록하여 확인 가능
특정 조도 값 이상이 되면 커튼이 자동으로 걷는 시스템
